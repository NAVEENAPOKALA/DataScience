def task_1 (df):
    columns_1=['views_day1','likes_day1', 'dislikes_day1', 'comments_day1', 'likes_ratio', 'comments_ratio']
    mean=df[columns_1].mean()
    median=df[columns_1].median()
    variance=df[columns_1].var()
    skw=df[columns_1].skew()
    krt=df[columns_1].kurtosis()

    final_result= pd.DataFrame ({'mean': mean, 'median': median,'variance': variance, 'skew': skw,'kurtosis': krt})
    return final_result

Selected Feature: Views Day 1

Proposed Distribution: Gamma Distribution

Reasoning:

  1. Views data is strictly positive (Gamma supports positive values only)
  2. The data is right-skewed with a heavy tail (Gamma can model this)
  3. Real-world view counts often follow power-law or Gamma-like distributions
  4. Gamma distribution is flexible with shape and scale parameters

import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns

from scipy.stats import norm, poisson, expon, gamma
import warnings
warnings.filterwarnings('ignore')

# Fit Gamma distribution to the data
params = gamma.fit(views_data)
x = np.linspace(views_data.min(), views_data.max(), 1000)
pdf_fitted = gamma.pdf(x, *params)

# Plot
plt.figure(figsize=(12, 6))

# Histogram
plt.hist(views_data, bins=50, density=True, alpha=0.7, color='skyblue', label='Data Histogram')

# Fitted distribution
plt.plot(x, pdf_fitted, 'r-', linewidth=2, label=f'Fitted Gamma Distribution')

plt.xlabel('Views Day 1', fontsize=12)
plt.ylabel('Density', fontsize=12)
plt.title('Task 2b - Distribution Fitting for Views Day 1\nGamma Distribution Fit', fontsize=14)
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()

# Goodness of fit assessment
print("\nGoodness of Fit Assessment:")
ks_stat, ks_pvalue = stats.kstest(views_data, 'gamma', args=params)
print(f"Kolmogorov-Smirnov Test: statistic = {ks_stat:.4f}, p-value = {ks_pvalue:.4f}")

# Calculate mean and variance of fitted distribution
shape, loc, scale = params
fitted_mean = shape * scale + loc
fitted_var = shape * scale**2

print(f"\nData Mean: {views_data.mean():.2f}, Fitted Gamma Mean: {fitted_mean:.2f}")
print(f"Data Variance: {views_data.var():.2f}, Fitted Gamma Variance: {fitted_var:.2f}")

Sample Variance Variability: The sample variances show much greater variability (23-38% differences from population variance). This is expected because: Variance is more sensitive to outliers than the mean. The highly skewed distribution with extreme outliers makes variance estimation challenging. Small samples may miss or over-represent extreme values.

Sampling Implications: For highly skewed distributions like video views, larger sample sizes would be needed for stable variance estimates. The mean is relatively more stable across samples compared to variance. The Law of Large Numbers suggests that with larger samples, these estimates would converge to the population parameters.
